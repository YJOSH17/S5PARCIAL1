ETIQUETAS META EN HTML

Las etiquetas <meta> en HTML proporcionan información adicional sobre el contenido de una página web. Se colocan dentro de la sección <head> y no se muestran en la página.

Ejemplos de Etiquetas <meta>

1. Codificación de Caracteres

Esta etiqueta define el conjunto de caracteres que el navegador debe usar para mostrar el contenido:

<meta charset="UTF-8">

Campo: Define la codificación de caracteres como UTF-8.

2. Vista en Dispositivos Móviles

Esta etiqueta asegura que la página se adapte correctamente a diferentes tamaños de pantalla:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Campo: Configura la vista para que se ajuste al ancho del dispositivo.

3. Descripción de la Página

Esta etiqueta proporciona una breve descripción del contenido de la página para los motores de búsqueda:

<meta name="description" content="Una breve descripción de la página">

Campo: Describe el contenido de la página para los motores de búsqueda.

4. Palabras Clave

Esta etiqueta especifica palabras clave relacionadas con el contenido de la página:

<meta name="keywords" content="HTML, etiquetas, meta">

Campo: Proporciona palabras clave para mejorar el SEO.

5. Autor de la Página

Esta etiqueta indica el autor del contenido de la página:

<meta name="author" content="Nombre del autor">

Campo: Define el autor del contenido.
