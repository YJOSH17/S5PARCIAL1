FORMULARIOS EN HTML

Los formularios en HTML son una manera de recopilar datos de los usuarios en una página web. Un formulario puede contener varios tipos de controles, como campos de texto, botones de opción, casillas de verificación, menús desplegables y botones de envío. Los formularios son esenciales para interactuar con los usuarios, permitiendo el envío de datos al servidor para su procesamiento.

A continuación, te mostramos algunos ejemplos básicos de cómo se crean diferentes elementos de un formulario en HTML.

Campo de texto

Este es un campo donde el usuario puede escribir información:

<input type="text" name="nombre" placeholder="Escribe tu nombre">

Botón de opción (radio button)

Permite seleccionar una opción entre varias:

<input type="radio" name="genero" value="masculino"> Masculino

Masculino
Femenino

Casilla de verificación (checkbox)

Permite seleccionar una o varias opciones:

<input type="checkbox" name="intereses" value="musica"> Música

Música

Menú desplegable

Permite seleccionar una opción de una lista:

<select name="pais"><option value="mexico">México</option><option value="espana">España</option></select>

Botón de envío

Permite enviar el formulario:

<input type="submit" value="Enviar">
