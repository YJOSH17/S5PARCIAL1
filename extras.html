ETIQUETAS EXTRAS Y OTRAS EN HTML

En HTML, existen varias etiquetas que son útiles para agregar estilos, scripts, y otros elementos adicionales a tu página web. Aquí te explicamos algunas de las más comunes:

1. La Etiqueta <style>

La etiqueta <style> se utiliza para agregar estilos CSS directamente dentro de un documento HTML:

<style>
    body {
        background-color: lightblue;
    }
    h1 {
        color: navy;
        font-family: Arial, sans-serif;
    }
</style>

Campo: Define estilos CSS dentro del HTML para personalizar la apariencia de los elementos en la página.

2. La Etiqueta <script>

La etiqueta <script> se usa para incluir código JavaScript en una página web, permitiendo agregar interactividad:

<script>
    function mostrarMensaje() {
        alert("¡Hola, bienvenido a mi página!");
    }
</script>

Campo: Incluye código JavaScript para realizar acciones dinámicas como mostrar un mensaje de alerta.

3. La Etiqueta <link>

La etiqueta <link> se utiliza para vincular archivos externos, como hojas de estilo CSS:

<link rel="stylesheet" href="styles.css">

Campo: Conecta un archivo CSS externo con la página HTML para aplicar estilos.

4. La Etiqueta <meta>

Ya hemos hablado de la etiqueta <meta>, pero es importante recordar que se usa para especificar metadatos sobre la página, como la codificación de caracteres o la descripción:

<meta charset="UTF-8">

Campo: Define la codificación de caracteres del documento como UTF-8.

5. La Etiqueta <base>

La etiqueta <base> se utiliza para definir una URL base para todos los enlaces relativos en la página:

<base href="https://www.ejemplo.com/">

Campo: Establece la URL base para todos los enlaces de la página.
